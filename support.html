WarpFetch Support

Help for WarpFetch

WarpFetch is a native macOS transfer manager for direct file links. If you need help with setup, purchases, browser handoff, or a transfer issue, use the contact details below.

Contact

  • Email: warpdownloader@gmail.com
  • Privacy Policy: warpdownload.github.io/warpdownload-site/privacy-policy.html
Include the app version and a short note about what you were trying to transfer so support can help faster.

Common Questions

  • WarpFetch is built for direct file URLs, not website scraping or media extraction.
  • Browser handoff is optional and can be enabled from Settings > Browser.
  • Subscriptions, free trials, restores, and lifetime unlocks are handled through Apple’s App Store systems.
  • You can manage or cancel subscriptions from your Apple account subscription settings.

Before You Email

  • Include the app version shown in Settings > Support.
  • Describe the direct file URL host or download source you are using.
  • Say whether the issue happens with a normal pasted URL, browser handoff, or batch import.
